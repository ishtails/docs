## Product

Verity is Proof-of-Value infrastructure for consultations.

It replaces time-based trust with outcome-based settlement by combining:
- pre-funded escrow,
- autonomous session evidence capture,
- AI-based merit scoring,
- on-chain payout/refund execution.

## Problem

Traditional consulting has a trust gap:
- Experts worry about payment risk.
- Learners worry about paying for low-value sessions.
- Outcome quality is difficult to verify objectively.

## Solution

Verity enforces a shared pact:
1. Define weighted success goals before the session.
2. Lock funds in escrow.
3. Capture objective meeting evidence.
4. Score delivery against goals.
5. Settle payout/refund with transparent logic.

## MVP scope

- Wallet-based access
- Host session creation (details -> goals -> share)
- Learner join + escrow funding
- Live session status
- Evaluation + result views
- Claim/dispute entry points
