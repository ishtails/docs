---
title: "Quickstart"
description: "Run and review the Verity demo flow"
---

## 1. Start the web app

```bash
cd packages/web
bun run dev
```

## 2. Open the end-to-end demo

Visit these routes in order:

1. `/demo`
2. `/demo/login`
3. `/demo/onboarding`
4. `/demo/host/dashboard`
5. `/demo/host/create/details`
6. `/demo/host/create/goals`
7. `/demo/host/create/share`
8. `/demo/learner/join`
9. `/demo/learner/dashboard`
10. `/demo/session/live`
11. `/demo/session/evaluating`
12. `/demo/session/results`

## 3. Use the docs set

- Product docs: `/verity/overview`
- Architecture: `/verity/architecture`
- Hackathon brief base: `/verity/hackathon-submission`

<Note>
  The demo flow is UI-only mock data for product iteration. Settlement and API behavior are represented visually.
</Note>
