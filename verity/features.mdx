## Smart escrow

- Learner locks USDC before session start.
- Funds remain rule-bound by contract logic.
- Settlement is executed after evaluation.

## Session evidence

- Recall.ai bot joins sessions automatically.
- Transcript and metadata are captured.
- Evidence can be persisted to decentralized storage.

## Merit evaluation

- Session is scored against weighted goals.
- Criteria includes relevance, depth, engagement, and clarity.
- Confidence and learning scores drive settlement.

## Merit settlement

Formula:

```txt
((confidence + learning) / 2)^2
```

- Produces teacher payout and learner refund split.
- Result is transparent and auditable.

## Dispute and finality

- Optional dispute window after evaluation.
- Automatic finalization after dispute window closes.

## Product surfaces

- Login and onboarding
- Host dashboard and setup flow
- Learner join and learner dashboard
- Live session state
- Evaluation and results
