## 1. Start the Web App

Verity's web layer is built with Bun and Hono.

```bash
cd packages/web
bun install
bun run dev
```

## 2. Explore the Demo Flow

The demo provides a UI-only walkthrough of the Verity experience. Visit these routes in order to see the full lifecycle:

1. **Onboarding**: `/demo/onboarding`
2. **Teacher Dashboard**: `/demo/host/dashboard`
3. **Create Session**: `/demo/host/create/details` → `/demo/host/create/goals`
4. **Learner Join**: `/demo/learner/join`
5. **Session Lifecycle**: `/demo/session/live` → `/demo/session/evaluating` → `/demo/session/results`
